# Lab-5-Server-Side-Rendering

Estructura
/server
   main.go
   db.go
   handlers.go
/static
   style.css
   app.js
/templates
   index.html
   create.html
series.db

Funcionalidades
- Listar series desde SQLite
- Crear serie con formulario (POST)
- Redirección 303 (POST/Redirect/GET)
- Incrementar episodio con botón +1 usando fetch()
- Tabla dinámica generada desde DB
- Archivos estáticos (CSS/JS)
- Manejo de errores
- UTF-8 configurado correctamente

Challenges implementados
- CSS / estilos
- Código Go organizado en archivos
- JS separado en archivo
- Barra de progreso
- Marcar serie completada
- Botón +1 con fetch()
- Servir estáticos
- Manejo correcto de DB
